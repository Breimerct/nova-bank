## 🇪🇸 NovaBank - Aplicación Bancaria

**NovaBank** es una aplicación web bancaria moderna para autenticación de usuarios, visualización de saldo, gestión de transferencias y filtrado avanzado de transacciones.

### ✨ Características

- ✅ Autenticación segura de usuarios
- 💰 Dashboard con visualización de saldo
- 📊 Historial completo de transferencias
- 🔄 Crear y gestionar transferencias
- 🔍 Filtrado avanzado de transacciones
- 🎨 Interfaz responsiva con Tailwind CSS
- 🌙 Soporte para tema oscuro/claro
- 🧪 Suite completa de tests (39 tests)

### 🚀 Inicio Rápido

```bash
# Instalar dependencias
pnpm install

# Ejecutar en desarrollo
pnpm dev

# Compilar para producción
pnpm build

# Ejecutar tests
pnpm test
```

### 🏗️ Stack Tecnológico

- **Frontend**: React 19.2.6 + TypeScript 6
- **Build**: Vite 8
- **Estilos**: Tailwind CSS 4
- **UI**: shadcn/ui
- **Estado**: Zustand 5
- **Formularios**: React Hook Form 7
- **Validación**: Yup 1
- **Testing**: Jest + React Testing Library
- **HTTP**: Axios 1

### 📦 Scripts Disponibles

```bash
pnpm dev              # Desarrollo
pnpm build            # Build producción
pnpm test             # Tests
pnpm test:watch       # Tests con watch
pnpm lint             # ESLint
pnpm format           # Prettier
pnpm typecheck        # TypeScript check
```
